\# Mini SOC Project using Wazuh



\## Overview

This project demonstrates a basic Security Operations Center (SOC) setup using Wazuh SIEM.



\## Setup

\- Wazuh Server (VM)

\- Ubuntu Agent



\## Attacks Simulated

\- SSH brute force

\- HTTP flood (DDoS simulation)

\- File integrity monitoring



\## Tools Used

\- Wazuh

\- Ubuntu

\- Kali Linux



\## Outcome

Successfully detected and analyzed security events using Wazuh dashboard.
