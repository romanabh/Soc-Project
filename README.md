# Mini SOC Project using Wazuh

## Overview
Built a Mini SOC lab using Wazuh SIEM to monitor and detect security events.

## Attacks Simulated
- SSH brute force
- HTTP flood (DDoS simulation)
- File integrity monitoring

## Tools Used
- Wazuh
- Ubuntu
- Kali Linux

## Outcome
Detected and analyzed attacks using Wazuh dashboard.
